HTML Notes




- HTML stand Hyper Text Markup Language.
- HTML is collection of Tags : Which are Instructions to the Browser.
- HTML is composed of three parts:
 1. HTML version information.
 2. Header Section.
 3. Body Section.

<!doctype html> <html>  <!-- Comment : Header Section -->  <head>   <!-- Comment : Title Section -->   <title>    This section contains HTML Title   </title>  </head>  <!-- Comment : Body Section -->  <body>   This section contains HTML Body  </body> </html>


doctype tag : mentions it is HTML5 Standard.
html tag : Main tag of HTML document.
head tag : Contains metadata info about the web page that are not displayed for the user.
title tag : Specifies the title of a web page. Title tags are displayed on search engine results pages (SERPs) as the clickable headline.
body tag : Defines the main content of an HTML document which displays on the browser.
<!-- --> : Defines comment tag, text written inside it, will not be processed by the browser, even if it is another HTML tag.

- HTML ignores empty spaces that are part of the document.
- HTML tags are Case Insensitive.

h1 to h6 tags : Used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.
p tag : Defines a paragraph. Browsers automatically add a single blank line before and after each p tag.
br tag : Inserts a single line break.
b tag : Specifies bold text.
i tag : Specifies italic text.

- Other text basics/formatting tags.
section tag : Represents a generic standalone section of a HTML document.
aside tag : Represents a portion of a document whose content is only indirectly related to the document's main content.
article tag : Represents a self-contained composition in a document.
blockquote tag : Specifies a section that is quoted from another source.

About Lists

li tag : Used to represent an item in a list. It must be contained in a parent element: an ordered list ( <ol> ) and an unordered list ( <ul> ).

<b> Ordered List </b> <ol>  <li> First Step </li>  <li> Second Step </li>  <li> Third Step </li>  <li> Fourth Step </li> </ol> <b> Ordered List type - Alphabets</b> <ol type="a">  <li> First Step </li>  <li> Second Step </li>  <li> Third Step </li>  <li> Fourth Step </li> </ol>


ol tag : Defines an ordered list. An ordered list can be numerical or alphabetical. type inside ol tag is an attribute and the listing can be changed to alphabets fashion.

<b> UnOrdered List </b> <ul>  <li> First Step </li>  <li> Second Step </li>  <li> Third Step </li>  <li> Fourth Step </li> </ul>


ul tag : Defines an unordered (bulleted) list.

<b> Definition List </b> <dl>  <dt> Topic One : </dt>  <dd> Description One </dd>  <br>  <dt> Topic Two : </dt>  <dd> Description Two </dd> </dl>


dl tag : Represents a description list. The element encloses a list of groups of terms (specified using the <dt> element) and descriptions (provided by <dd> elements).

About Image

img tag : Used to embed an image in an HTML page. Images can be added from Locally or from link. src attribute holds the value of the image's location. width and height attribute will help us to resize the image to our need. alt attribute helps us to have an alternate text when image is not available.

figure tag : Specifies self-contained content. Content of this tag is related to the main flow, its position is independent of the main flow, and if removed it should not affect the flow of the document.
figcaption tag : Defines a caption for a figure tag.

About CSS

- CSS is the language used to style a Web page.
- CSS stands for Cascading Style Sheets.
- CSS works on the presentation aspect of the webpage.
- 3 Different ways you can mention the styles: From Lower Priority to Higher Priority.
 1. From file with the extension *.css
  Snippet :

   <link href="./style.css" rel="stylesheet" type="text/css">

  href attribute : location of the file present.
  rel attribute : relationship attribute tells how this document should be treated.
  type attribute : tells what kind of document.

 2. With style tag.
  Snippet :

   p {
    text-align: right;
   }

 3. With Style attribute.
  Snippet :

   <p style="text-align: left;"> This text will be left aligned </p>

Common Styles :

1. text-align : Text aligned either center, right or left. Used with heading tags, paragraph tags etc.
2. font-size : xx-small, x-small, small, medium, large, x-large and xx-large are the font sizes available. Used with heading tags, paragraph tags etc. YOu can also provide absolute size using %, px, cm and in. Example :- <p style="font-size: 40px;"> Check This One! </p>
3. font-family : Text font style/s will be changed with the help of this property in the style attribute.
4. color : Text color can be changed with this property in the style attribute. Value takes names, hsl, rgb method(Absolute values for 3 colors) and also Hexa Decimal Values(Starts with '#' symbol) for the shaded colors.
5. list-style-type : Used with OrderedList or UnOrderedList. List style can be changes from number to characters to images. It allows Custom images as List indexes.
6. font-style : Text style can be changed with this - bold, italic and underline.

And many more properties are present only obvious ones are discussed here.

Links to other pages :

 Snippet :

  <a href="www.google.com" target="_blank"> Google Me! </a>

a tag : Anchor tag, it anchors / navigates to a different page on clicking the mentioned text of a tag. href can also contain another html files which are locally available. Target attribute mentions whether to navigate to a page in the same window or different window.
nav tag : Used for declaring the navigational section in HTML documents. Defines a set of navigation links.

div tag : Defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. Properties like Width, Height, Margin, Padding and Background color can be manipulated with CSS.

Tables in the webpage

<table>  <thead>   <tr>    <th>First Name</th>    <th>Last Name</th>   </tr>  </thead>  <tbody>   <tr>    <td>Name 1</td>    <td>Last 1</td>   </tr>   <tr>    <td>Name 2</td>    <td>Last 2</td>   </tr>  </tbody>  <tfoot>   <tr>    <td>Name 3</td>    <td>Last 3</td>   </tr>  </tfoot> </table>


table tag : Defines HTML table. HTML table consists of one or more tr tag, th tag, and td tag elements.
th tag : Defines a header cell in an HTML table.
tr tag : Defines a row of cells in a table.
td tag : Defines a standard data cell in an HTML table.
thead tag : Used to group header content in an HTML table.
tbody tag : Used to group the body content in an HTML table.
tfoot tag : Used to group footer content in an HTML table.

User Input Form in webpage

form tag - used to create an HTML form for user input.
input tag - Individual User Input type.
 Snippet :

   <input type="text" name="username" id="username" required>

 Important attributes:
  type - Used for what type of User Inout to expect. Different types are - text, number checkbox, radio, email, password, date, time etc
  required - Used when the Input tag made mandatory to proceed to next navigation/submission. By default, input is not mandatory.
